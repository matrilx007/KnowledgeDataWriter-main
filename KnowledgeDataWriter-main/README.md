# 数据录入平台

一个基于 React + TypeScript + TailwindCSS 开发的纯前端数据录入平台，支持 QA 问答式录入和分段式录入，使用浏览器本地存储保存数据。

## 功能特性

### 🔹 QA 问答式录入
- 预设 10 个常见问题
- 逐题作答，支持前后导航
- 实时进度显示
- 答案自动保存到本地存储

### 🔹 分段式录入
- 粘贴长文本自动分段
- 支持段落编辑、删除、新增
- 实时数据保存

### 🔹 数据管理
- **自动保存**：每次修改实时保存到 localStorage
- **数据恢复**：页面刷新后自动加载上次数据
- **文件导出**：一键导出为 .txt 文件
- **数据清空**：支持清空所有本地数据

## 快速开始

### 安装依赖
```bash
pnpm install
```

### 启动开发服务器
```bash
pnpm run dev
```

### 构建生产版本
```bash
pnpm run build
```

## 使用说明

### QA 录入模式
1. 点击导航栏的"QA 录入"
2. 在答案输入框中填写回答
3. 使用"上一题"/"下一题"按钮导航
4. 进度条显示完成情况
5. 点击"导出数据"下载 .txt 文件

### 分段录入模式
1. 点击导航栏的"分段录入"
2. 在文本输入框中粘贴或输入内容
3. 点击"添加分段"自动分割段落
4. 对每个段落进行编辑或删除
5. 点击"导出数据"下载 .txt 文件

## 数据格式

### QA 导出格式
```
Q1: 请输入您的姓名
A1: 张三

Q2: 请输入您的联系方式
A2: 13800138000
```

### 分段导出格式
```
段落1: 这是第一个段落的内容
段落2: 这是第二个段落的内容
段落3: 这是第三个段落的内容
```

## 技术栈

- **前端框架**：React 19 + TypeScript
- **样式方案**：TailwindCSS 4.x
- **构建工具**：Vite 7.x
- **包管理器**：pnpm
- **数据存储**：localStorage（浏览器本地存储）
